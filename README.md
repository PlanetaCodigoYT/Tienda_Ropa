👕 Proyecto: Tienda de Ropa

Este repositorio contiene el desarrollo paso a paso de una aplicación web funcional para la gestión de una tienda de ropa.

Contenido del repositorio:

codigo/: Lógica del sistema y vistas estructuradas.

base-de-datos/: Script .sql para recrear la estructura de tablas en MySQL.

documentacion/: Diapositivas explicativas del modelo de negocio y arquitectura.
